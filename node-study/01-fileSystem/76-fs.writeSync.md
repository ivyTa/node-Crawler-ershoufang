# node

## 1. fs 文件模块

### 1.76. fs.writeSync

#### fs.writeSync(fd, buffer[, offset[, length[, position]]])

- fs.write() 的同步版本。返回写入的字节数。

> Syntax

```js
//fd <integer>
//buffer <Buffer> | <Uint8Array>
//offset <integer>
//length <integer>
//position <integer>
fs.writeSync(fd, buffer[, offset[, length[, position]]])
```

#### fs.writeSync(fd, buffer[, offset[, length[, position]]])

- fs.write() 的同步版本。返回写入的字节数。

> Syntax

```js
//fd <integer>
//string <string>

fs.writeSync(fd, buffer[, offset[, length[, position]]])
```
